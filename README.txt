# Connect Intel Edison to Microsoft Azure Cloud and emit data from Arduino IDE

 - edison-sensor-read-demo.ino
  This Arduino sketch basically simulates a sensor reading which is printed on the serial monitor.

 - index.js
  To run the script, enter "node index.js <usb serial connection port>"  the connection port is something like /dev/usb.modem1a123

  Issues: Could not establish connection via Microsoft Azure IOT Hub
